## Getting Started | 开始

1. Pick the Phase repository you're working on | 找到你要上的 Phase 仓库
2. `git clone` to local | 克隆到本地
3. Create your own branch: `git checkout -b your-name` | 创建你自己的分支
4. Follow the `README.md` inside each day's folder | 每天打开对应文件夹的 README，按步骤操作
5. `git add -A && git commit -m "Day X done" && git push`

## Teaching Philosophy | 教学理念

**No leaderboard — you be the judge. From evaluation to deployment, walk through the full LLM engineering chain in days. Define your own theme — hook into the API, not the implementation.**

不看排行榜，自己动手做裁判。从评测到部署，走通大模型工程化全链路。主题你自己定——只抓接口，不抓实现。
