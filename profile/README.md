## 怎么开始？

1. 找到你要上的 Phase 仓库
2. `git clone` 到本地
3. 创建你自己的分支：`git checkout -b 你的名字拼音`
4. 每天打开对应文件夹的 `README.md`，按步骤操作
5. `git add -A && git commit -m "DayX done" && git push`
